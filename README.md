# 📚 Code to PDF Generator

## What is this?
This tool converts your entire project's source code into a beautiful, well-formatted PDF.

## How to use? (Very Easy)

### Step 1: Download the file
Download the `pdf-generator.js` file

### Step 2: Copy to your project
Copy `pdf-generator.js` into your **project root folder** (where your package.json is located)

### Step 3: Install dependency
Open terminal/command prompt and run:
```bash
npm install puppeteer-core

Step 4: Generate PDF

node pdf-generator.js
